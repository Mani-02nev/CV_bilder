## ðŸš€ How It Works

### **Auto-Cycling:**
```typescript
useEffect(() => {
    const interval = setInterval(() => {
        setIsTyping(true)
        setTimeout(() => {
            setStep((prev) => (prev + 1) % demoSteps.length)
            setIsTyping(false)
        }, 1500)
    }, 4000)
    
    return () => clearInterval(interval)
}, [])
```

### **Step Transition:**
1. **Typing starts** (1.5 seconds)
   - Icon rotates and scales
   - Pulsing dots appear
   - "AI is working..." message

2. **Content appears** (0.5 seconds)
   - Fade in animation
   - Slide up effect
   - Green checkmark

3. **Wait** (2 seconds)
   - Display complete content
   - User can read

4. **Next step** (repeat)
