## Typography

### **Modern:**
- Font: Sans-serif
- Name: 4xl (36px), bold
- Headers: xl (20px), bold, uppercase
- Body: sm (14px)

### **Classic:**
- Font: Serif
- Name: 3xl (30px), bold, uppercase
- Headers: lg (18px), bold, uppercase
- Body: sm (14px)

- Font: Sans-serif
- Name: 5xl (48px), light

- Body: xs (12px)

### **Creative:**
- Font: Sans-serif
- Name: 4xl (36px), bold
